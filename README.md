# ⚡ Electrostatics Field Line Simulator

An interactive, real-time 2D physics simulation built in Python and Pygame that visualizes continuous electric field lines generated by point charges.

## 🧠 The Physics & Mathematics

Unlike static textbook diagrams, this simulation dynamically generates streamlines based on the exact positional data and magnitude of the charges on screen.

  - Superposition Principle: The simulation calculates the net electric field vector E at any given point by calculating the vector sum of Coulomb's Law for every charge on the canvas.
  - Euler Integration: The algorithm calculates the normalized net field, takes a discrete spatial step  \(\Delta \)s in that direction, and repeats the process to trace the continuous streamline.

## 🎮 How to Use / Controls

The canvas is fully interactive. You can add, remove, and manipulate charges in real-time to see how the field lines instantly recalculate and snap into new equilibrium states.

**Action**                            **Control**
Create a New Charge               Left Click on empty space
Move a Charge                     Left Click + Hold on a charge and drag
Change Magnitude                  Left Click + Hold + Scroll Wheel on a charge
Remove a Charge                   Right Click on a charge

*(Note: Charges start with a default magnitude of 40 C. Scrolling up increases the positive charge, scrolling down decreases it into a negative charge).*

# 🛠️ Tech Stack & Components

  - Language: Python.
  - Graphics: pygame (Handling the event loop, rendering shapes, and drawing anti-aliased line segments).
  - Mathematics: numpy (for angular distribution of seed points) and a custom Object-Oriented Vector class handling 2D vector addition, subtraction, scalar multiplication, and normalization.
